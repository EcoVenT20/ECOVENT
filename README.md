# ECOVENT - Industrial Ventilation Solutions 🌀
**الريادة في حلول التهوية الصناعية - أنظمة متكاملة تجمع بين القوة والذكاء.**

## 📖 وصف المشروع (Description)
الموقع الرسمي لشركة **ECOVENT** المتخصصة في تصميم وتوريد أنظمة التهوية الصناعية المتطورة (Axial & Centrifugal Fans) وأنظمة التحكم الذكية في المملكة العربية السعودية. يهدف الموقع للتعريف بالهوية البصرية للشركة واستعراض الحلول التقنية الحديثة.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **Frontend:** HTML5, CSS3, JavaScript.
- **Design Framework:** Responsive Web Design (Flexbox / CSS Grid).
- **Assets:** Optimized Industrial Graphics & Motion Graphics Presentations.
- **Deployment:** Hostinger (Domain) & Private Server Integration.

## 📁 هيكلة الملفات (File Structure)
- `index.html`: حجر الأساس والصفحة التعريفية الرئيسية.
- `style.css`: التنسيقات البصرية والهوية اللونية للشركة (الأزرق والأخضر).
- `assets/`: مجلد الملحقات (الصور، الفيديوهات، والأيقونات).
- `js/`: ملفات التفاعل البرمجية وتنسيق تجربة المستخدم.

## 🚀 المميزات التقنية (Technical Features)
- **Responsive Layout:** الموقع متوافق تماماً مع جميع الشاشات (Desktop, Tablet, Mobile).
- **SEO Optimized:** هيكلة الأوسمة (Semantic HTML) مُعدة لمحركات البحث لضمان ظهور المنتجات.
- **Performance:** ضغط الصور والملفات لضمان سرعة التحميل العالية.
- **Interactive UI:** استخدام الـ Transitions والـ Hover effects لتحسين تجربة العميل.
